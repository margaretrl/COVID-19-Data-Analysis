Git Commands
(don't include brackets when inputting)

Cloning Repository: 
git clone [link_to_repo]

Changing Branches:
git checkout [branch_name]

Creating a New Branch:
git checkout -b [new_branch_name]
    Note: it won't show up until you push it

Checking Branch info:
git status

To see list of all Branches:
git branch -a 

To Push Changes to Repo:
git add -u  //adds all files to be commited 
git status  //make sure all files were added correctly
git commit -m "Brief message about what you're commiting"
git push 
git status  //make sure there is nothing left to push 

To Update Branch:
git fetch
git pull
